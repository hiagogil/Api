<div class="pilwrap" id="pre-requisito">
  <h3>
    <a href="#pre-requisito" name="pre-requisito" class="pilcrow"></a>
Pré-requisito:
  </h3>
</div>
<p>-Node.js - Você encontra em: https://nodejs.org/en/</p>
<p>-Visual Studio Code ou qualquer editor de texto - você encontra em: https://code.visualstudio.com/download</p>
<p>-Git: você encontra em: https://git-scm.com/downloads</p>
<p>Via terminal, rode os seguintes comandos:</p>
<pre><code>git clone https://github.com/EBAC-QE/testes-api-cy.git
</code></pre>
<pre><code>cd testes-api-cy
</code></pre>
<div class="pilwrap" id="para-instalar-as-dependencias">

</code></pre>

</code></pre>

</code></pre>

</div>

</code></pre>
